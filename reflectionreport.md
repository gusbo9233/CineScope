# Reflection Report

During this project, I learned how to build a web application using the ASP.NET Core MVC pattern. I got more familiar with separating the application into models, views, controllers, and services, and I saw how this structure makes the project easier to understand and maintain.

I also learned how to use Entity Framework Core to connect the application to a database. This included creating models, using a database context, working with migrations, and storing data such as movies, ratings, comments, and favorites.

Another important part of the project was learning about authentication and authorization. I used Microsoft Identity Web and role-based access so that regular users and admin users could access different parts of the application.

The project also introduced me to using an external API through the OMDb API. I learned how to fetch movie data, map API responses into local models, and save selected movies into the database.

Finally, I learned more about cloud deployment and Azure hosting. The project includes Azure App Service support, SQL Server configuration, Bicep infrastructure, and GitHub Actions for deployment. This helped me understand how a local MVC application can be prepared for hosting in the cloud.

Overall, CineScope helped me practice full-stack web development with ASP.NET Core, Entity Framework, authentication, external APIs, and Azure deployment.
